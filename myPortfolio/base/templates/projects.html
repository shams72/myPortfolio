TravelAgency

As part of a university project, I developed a comprehensive application utilizing QtCreator. This application seamlessly manages customer data, enabling seamless reading, insertion, and deletion of information stored in a JSON file. Booking an object within the system involves capturing essential customer details, and dynamically generating unique identification for each booking. Additionally, the application integrates a map feature, offering customers a visual representation of their journey from the starting point to their final destination.
